# Desktop Buddy Roadmap

Our vision is to build the most delightful and effective wellness reminder app for developers and knowledge workers.

## Current Status
- **v1.0.0** (March 2026) - Initial public release with core features and dual themes

---

## 🚀 Planned Features

### Phase 2: Analytics & Insights (Q2 2026)
- [ ] **Mood & Energy Trends** - Visual charts showing daily mood/energy patterns
- [ ] **Productivity Insights** - Weekly/monthly summaries of completed tasks
- [ ] **Streak Tracking** - Maintain streaks for habits (e.g., daily check-ins)
- [ ] **Data Export** - Export activity history to CSV/JSON for personal analysis
- [ ] **Activity Dashboard** - Advanced statistics page with interactive charts

### Phase 3: Advanced Customization (Q3 2026)
- [ ] **Custom Reminders** - User-defined reminder categories and schedules
- [ ] **Notification Sounds** - Optional audio alerts for reminders
- [ ] **Theme Customization** - Allow users to create custom color themes
- [ ] **Localization** - Multi-language support (Spanish, French, German, etc.)
- [ ] **Keyboard Shortcuts** - Global hotkeys for quick actions

### Phase 4: System Integration (Q4 2026)
- [ ] **Startup on Login** - Auto-launch app with OS startup
- [ ] **System Tray Minimization** - Better tray integration
- [ ] **Desktop Notifications** - Native OS notifications (Windows/macOS/Linux)
- [ ] **Calendar Integration** - Sync with Outlook/Google Calendar

### Phase 5: Cross-Platform (2027)
- [ ] **macOS Support** - Native macOS build and App Store presence
- [ ] **Linux Support** - Linux distributions support
- [ ] **iOS Companion App** - Mobile check-in and notifications
- [ ] **Web Dashboard** - Cloud-synced web interface

### Phase 6: Data & Privacy (2027)
- [ ] **SQLite Migration** - Local database instead of JSON files
- [ ] **Encryption** - End-to-end encryption for sensitive data
- [ ] **Data Backup** - Automatic local backups and recovery
- [ ] **Privacy Settings** - Granular control over data collection

---

## 🎯 Long-Term Vision

- **AI-Powered Insights** - Machine learning suggestions for optimal break times
- **Team Features** - Share wellness goals with colleagues (opt-in)
- **API & Plugins** - Extensibility for custom integrations
- **Browser Extension** - Lightweight reminders while working in web apps
